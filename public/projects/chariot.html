Company Purpose

Chariot is an integrated tool which allows nonprofits to easily receive donations through Donor Advised Funds, or DAFs. This tool can come as popup widget, an iframe, a weblink, a script-insertable button, or as an integration through another third-party donation platform such as Givebutter or Blackbaud.

Chariot Links

Chariot Links is the product which allows nonprofits to utilize an iframe, weblink, QR code, or embeddable button to display Chariot's product. It is hosted on the Chariot website and can be linked from anywhere to open a donation page where a user can use their DAF to donate to that nonprofit.

Summary

  • Created 3 screens to search for DAFs, add donor information, and confirm that the donation was successfully sent
  • Modified the existing Chariot Connect package to allow the Links pages to integrate smoothly with Connect
  • Developed an embeddable button which opens a Chariot Links popup, which nonprofits can utilize using a script tag to add to their website

Blackbaud Integration

Blackbaud is a ubiquitous donation platform used by many nonprofits around the world. Nonprofits may choose to add third-party widgets to their donation forms in order to give their donors more opportunities and methods to donate.

Summary

  • Developed and designed an integration with the Blackbaud platform as an add-in for nonprofits to add to their donation forms
  • Created a new data type to hold integration information and access points for the integration to utilize
